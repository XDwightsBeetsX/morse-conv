## Character Support

| Category | Support |
| :-- | :-- |
| English Characters | `[A - z]` |
| English Symbols | `[. , ; : ! ?]` |
| Numbers | `[0-9]` |
| Math | `[+ - / =]` |
| Currency | `[$]` |
| Unsupported | `[#]` |
